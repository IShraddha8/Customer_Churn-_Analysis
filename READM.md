 Customer Churn Analysis

# Overview
A Customer Churn Analysis project using SQL, Python, Power BI, and XGBoost to identify customers at risk of leaving and generate business insights.

# Tools & Technologies
- SQL Server
- Python
- Pandas
- Power BI
- XGBoost
- Scikit-learn

# Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- Customer Churn Prediction using XGBoost
